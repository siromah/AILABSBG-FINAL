# Media Assets

Place background media files here for use with the `BackgroundMedia` component.

## Expected files

- `learning-bg.mp4` — Main background video (homepage hero or about)
- `learning-bg.webm` — WebM fallback for better compatibility
- `learning-bg.jpg` — Static image fallback (used when video fails or reduced motion is preferred)

## Notes

- Videos should be muted, looped, and subtle. Low contrast movement.
- Keep files under 5MB for fast loading.
- Use warm, human, non-corporate visuals.
- If no video is available, the component gracefully falls back to the image or a solid background.
